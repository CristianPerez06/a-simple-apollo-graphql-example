# a-simple-apollo-graphql-example
A minimal example on how to use GraphQL with a ReactJS app and a NodeJS server.

### Frontend

- Javascript
- React
- Apollo
- GraphQL

### Backend

- Javascript
- Node
- Express
- Apollo server
- GraphQL

## How to execute the app

1. Go to /server
2. `npm install`
3. `npm run server`
4. Go to /app
5. `npm install`
6. `npm start`
